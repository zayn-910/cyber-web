Common Cyberattacks

1.Phishing

Fake emails or messages trick you into sharing personal info.

Precautions:

  • Check sender email addresses
  • Avoid clicking unknown links
  • Use email filters

2.Malware

Malicious software harms your device or steals data.

Precautions:

  • Install antivirus software
  • Update your system regularly
  • Avoid suspicious downloads

3.DDoS Attack

Overwhelms a website with traffic to shut it down.

Precautions:

  • Use a firewall
  • Monitor website traffic
  • Enable rate limiting

4.Password Attack

Hackers guess or steal passwords to gain access.

Precautions:

  • Use strong, unique passwords
  • Enable two-factor authentication
  • Change passwords regularly

5.SQL Injection

Inserts malicious code into databases to steal data.

Precautions:

  • Use prepared statements
  • Validate user inputs
  • Limit database permissions

6.Cross-Site Scripting (XSS)

Injects malicious scripts into websites to harm users.

Precautions:

  • Sanitize user inputs
  • Use Content Security Policy
  • Escape user data

7.Man-in-the-Middle (MITM)

Intercepts communication to steal data.

Precautions:

  • Use HTTPS websites
  • Avoid public Wi-Fi
  • Use a VPN

8.Ransomware

Locks your files and demands payment for access.

Precautions:

  • Back up data regularly
  • Avoid unknown email attachments
  • Update software

9.Social Engineering

Manipulates people into revealing sensitive info.

Precautions:

  • Verify requests for info
  • Educate employees
  • Be cautious with strangers

10.Credential Stuffing

Uses stolen credentials to access other accounts.

Precautions:

  • Use unique passwords
  • Monitor account activity
  • Enable login alerts
